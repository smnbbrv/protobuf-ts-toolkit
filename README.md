# protobuf-ts-toolkit

A collection of utilities and interceptors for [protobuf-ts](https://github.com/timostamm/protobuf-ts).

## Packages

| Package | Description |
|---------|-------------|
| [@protobuf-ts-toolkit/grpc-error](./packages/grpc-error/) | Error classes, interceptor, and codec for Google RPC error details |
| [@protobuf-ts-toolkit/metadata](./packages/metadata/) | Metadata interceptor for headers, tokens, and dynamic values |
| [@protobuf-ts-toolkit/opentelemetry](./packages/opentelemetry/) | OpenTelemetry tracing interceptor |
| [@protobuf-ts-toolkit/grpc-web-hybrid](./packages/grpc-web-hybrid/) | HTTP + WebSocket hybrid transport for full streaming support |
| [@protobuf-ts-toolkit/browser-logger](./packages/browser-logger/) | Browser console logging interceptor |
